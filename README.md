# Smart Contracts for lokkit

### Install required packages
```
npm install -g truffle
npm install -g ethereumjs-testrpc
```

### Start testrpc
This will start a test node (listening on localhost:8545).
```
testrpc
```

### Build contracts
```
truffle build
```

### Run tests
```
truffle test
```

### Deploy contracts to localhost
```
truffle test
```
